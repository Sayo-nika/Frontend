# Sayonika (Frontend)

The frontend for the Sayonika DDLC mod data base.

## Build Instructions

### Requirements:
- NodeJS (v8 or higher)
- Yarn
- TypeScript

Run the following commands to build and test the Frontend:

```
yarn install
yarn run dev
```


To compile as a site for distribution:
```
yarn run build
```
